Server security random-walk simulation

This interactive simulation models a server that receives weekly security updates for n weeks.
Each week there are m independent attackers.
Each attacker succeeds in breaching with probability p. A week is considered "secure" (score +1) when no attacker succeeds that week; otherwise it is "breached" (score -1). Over n weeks the total score for a trajectory is the sum of weekly +1 / -1 outcomes. The number of secure weeks K is binomial(n, q) with q = (1-p)^m, so the total score S = 2K - n.
