DATASET

A dataset is an organized collection of data used for analysis or processing. It typically contains observations (rows) and features or variables (columns). Datasets are commonly used for descriptive statistics, training machine learning models, or storing information.

Key aspects to consider:

  • Structure: structured (tables/CSV), semi-structured (JSON, XML), or unstructured (text, images, audio).
  • Labels: some datasets include target values (for supervised learning), while others do not.
  • Metadata: schema, units, source, and collection date provide important context.
  • Quality: completeness, accuracy, consistency, and potential bias affect analysis results.
  • Privacy & ethics: sensitive data require anonymization and proper consent.

Simple example: a CSV file where each row represents a person and columns contain age, height, occupation, and optionally a "label" column for classification.

DISTRIBUTION

A distribution describes how the values of a variable are spread across possible outcomes. It tells you which values are common, which are rare, and how spread out the data are. Understanding a distribution is fundamental for summarizing data and making inferences.

Key points about distributions:

  • Shape: symmetric, skewed (left/right), uniform, or multimodal (multiple peaks).
  • Center: measures like mean, median, and mode indicate a typical value.
  • Spread: variance, standard deviation, and IQR describe variability.
  • Tails & outliers: extreme values can affect analysis and indicate special cases.
  • Types: discrete distributions (counts, e.g., Poisson) vs. continuous distributions (measurements, e.g., Normal).
  • Visualization: histograms, density plots, boxplots and empirical cumulative distribution functions help reveal distributional properties.

Simple example: human heights are roughly normally distributed (bell-shaped), while the number of emails received per day is often skewed and better modeled with a discrete distribution.

Caesar cipher

                        
The Caesar cipher is a simple substitution cipher that shifts each letter by a fixed number of positions (the shift). For example, with shift 3, A → D, B → E, etc. It preserves letter case and leaves non-letter characters unchanged.
